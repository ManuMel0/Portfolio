Quem sou

Sou apaixonada por tecnologia e pelo potencial dos dados para gerar insights e apoiar decisões estratégicas. Minha curiosidade pela capacidade de processamento das máquinas me levou a atuar na área de Engenharia de Dados, onde transformo grandes volumes de informação em ativos valiosos para o negócio.

Atualmente, trabalho com desenvolvimento de pipelines ETL, processamento de dados e soluções de Big Data utilizando Python, SQL, PySpark, Databricks e AWS. Sou certificada como Databricks Certified Data Engineer Associate e AWS Certified Cloud Practitioner.

Tenho como objetivo construir plataformas de dados escaláveis, automatizar processos e aplicar boas práticas de engenharia para entregar soluções confiáveis, eficientes e orientadas a dados.
